> **VELOX** is a premium footwear e-commerce platform I'm building in **three versions** to master full-stack and mobile development.

| Version | Stack | Status |
|---------|-------|--------|
| 🌐 **VELOX Web (Laravel)** | PHP · Laravel · MySQL · Blade Templates | 🔨 Active |

- Product catalog with categories, filters & search

- Responsive UI across web and mobile
